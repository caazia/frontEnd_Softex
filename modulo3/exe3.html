Para esvaziar completamente as três estruturas de dados (fila, lista e pilha) utilizando o array: 1- Fila: A fila segue a regra FIFO (First-In, First-Out), o primeiro elemento a ser inserido é o primeiro a ser removido. Portanto, a sequência dos números removidos para esvaziar a fila seria: 3, 7, 9, 1, 0. 2- Lista: A lista não possui uma regra específica para remoção de elementos. Portanto, podemos remover os elementos da lista na ordem em que foram inseridos. A sequência dos números removidos para esvaziar a lista seria: 3, 7, 9, 1, 0. 3- Pilha: A pilha segue a regra LIFO (Last-In, First-Out), o último elemento a ser inserido é o primeiro a ser removido. Portanto, a sequência dos números removidos para esvaziar a pilha seria: 0, 1, 9, 7, 3.
